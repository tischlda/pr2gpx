pr2gpx
======

Converts Winlink position sent or received with Airmail into GPX tracks or waypoints.


Developing
----------

Requires Ruby >= 1.9

Source: http://github.com/tischdla/pr2gpx


Installing
----------

Requires Ruby >= 1.9

    gem install pr2gpx


Using
-----

Examples:

    pr2gpx --help

Displays information about the usage.
